# Изображения карт

Поместите сюда изображения карт CS2 в формате JPG.

## Формат именования

Файлы должны называться по имени карты (как в данных):

- `de_dust2.jpg`
- `de_mirage.jpg`
- `de_inferno.jpg`
- `de_nuke.jpg`
- `de_ancient.jpg`
- `de_anubis.jpg`
- `de_vertigo.jpg`

## Рекомендации

- Размер: 64x64 или 128x128 пикселей
- Формат: JPG (для фото) или PNG (для скриншотов)
- Источник: скриншоты карт из CS2 или изображения с сайта FACEIT
